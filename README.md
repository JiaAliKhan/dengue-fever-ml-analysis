# Dengue Fever Analysis Using Machine Learning

## Project Overview
This project analyzes dengue fever cases using data science and machine learning techniques. The dataset contains information about dengue cases, deaths, regions, months, and years. The goal of the project is to understand patterns in dengue outbreaks and apply machine learning models to analyze and predict dengue cases.

## Dataset
The dataset includes the following columns:

Month – Month of reported dengue cases  
Year – Year of reported data  
Region – Region where the cases occurred  
Dengue_Cases – Number of dengue infections  
Dengue_Deaths – Number of deaths caused by dengue  

## Data Visualization
Several visualizations were created to understand the dataset:

- Dengue cases by region
- Dengue deaths by region
- Dengue cases over years

These visualizations help identify trends and patterns in dengue outbreaks.

## Machine Learning Models
The following models were used in this project:

- Linear Regression
- Decision Tree Regression
- Random Forest Regression

The models were trained to predict dengue cases using the dataset features.

## Model Comparison
The performance of the models was compared using Mean Squared Error (MSE). Lower error values indicate better prediction accuracy.
